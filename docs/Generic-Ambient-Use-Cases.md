## Summary of Conversation Patterns

### Effective Patterns Across All Use Cases:

1. **Start with User Data** - Reference specific books, patterns, dates
2. **Ask Clarifying Questions** - Understand context before suggesting
3. **Provide Evidence** - Quote their highlights, show patterns
4. **Offer Options** - Give 2-3 choices, let them pick
5. **Connect Dots** - Show relationships they haven't seen
6. **Take Action** - Create lists, save notes, set goals
7. **Follow-Up Hooks** - "Come back and tell me how it goes"
8. **Celebrate Progress** - Acknowledge growth and completion
